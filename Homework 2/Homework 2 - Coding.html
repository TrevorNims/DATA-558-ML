Derivations¶

In this first section, I will perform a couple of important derivations on our objective $F(\beta)$:

$$F(\beta) = \frac{1}{2n}\sum_{i=1}^{n}(y_i-x_i^T\beta)^2+\frac{\lambda}{n}||\beta||_2^2$$

First, under the assumption of $d = 1$ and $n = 1$, I will find the derivatie of our objective $F(\beta)$.

$$ \begin{align*} \frac{\partial}{\partial \beta}F(\beta) &= \frac{\partial}{\partial \beta}(\frac{1}{2}(y-x\beta)^2+\lambda \beta^2) \\ &= -x(y-x\beta) + 2\lambda \beta \end{align*} $$

Now, under the assumption of $d > 1$ and $n > 1$, I will find the derivatie of our objective $F(\beta)$.

$$ \frac{\partial}{\partial \beta}F(\beta) = \frac{\partial}{\partial \beta}(\frac{1}{2n}||Y-X^T\beta||_2^2)+\frac{\partial}{\partial \beta}(\frac{\lambda}{n}||\beta||_2^2) $$

Let us name the first addend $f(\beta)$ and the second $g(\beta)$, then we have: $$ \begin{align*} \frac{\partial}{\partial \beta}f(\beta) &= \frac{\partial}{\partial \beta}(\frac{1}{2n}(Y-X^T\beta)^T(Y-X^T\beta)) \\ &= \frac{\partial}{\partial \beta} (\frac{1}{2n}(Y^TY-Y^TX^T\beta-\beta^TXY+\beta^TXX^T\beta)) \\ &= \frac{1}{2n}(-\frac{\partial}{\partial \beta}(Y^TX^T\beta)-\frac{\partial}{\partial \beta}(\beta^TXY)+\frac{\partial}{\partial \beta}(\beta^TXX^T\beta)) \\ &= \frac{1}{2n}(-XY-XY + (XX^T + (XX^T)^T)\beta) \\ &= -\frac{X}{n}(Y-X^T\beta) \end{align*} $$ Now consider $g(\beta)$: $$ \begin{align*} \frac{\partial}{\partial \beta}g(\beta) &= \frac{\partial}{\partial \beta}(\frac{\lambda}{n}\beta^T\beta) \\ &= (\frac{2\lambda}{n})\beta \end{align*} $$ Therefore, the gradient is given by the sum $f(\beta)+g(\beta)$, that is: $$\frac{\partial}{\partial \beta}F(\beta) = -\frac{X}{n}(Y-X^T\beta)+(\frac{2\lambda}{n})\beta$$

# Define gradient function

def compute_grad(beta, X, y, lam):
    n = len(y)
    x_tb = np.dot(np.transpose(X), beta)
    return -1/n*np.dot(X, y-x_tb)+2*lam/n*beta
In [8]:
# Define gradient descent function

def grad_descent(beta_init, step_size, num_iter, *args):
    iterates = []
    beta = beta_init
    for i in range(num_iter):
        new_beta = beta - step_size*compute_grad(beta, args[0], args[1], args[2])
        iterates.append(new_beta)
        beta = new_beta
    return iterates

# Define objective function

def obj_fun(beta, X, y, lam):
    n = len(y)
    x_tb = np.dot(np.transpose(X), beta)
    return 1/2*n*np.dot(np.transpose(y-x_tb), y-x_tb)+lam/n*np.dot(np.transpose(beta), beta)

# Define gradient descent function with additional epsilon parameter

def grad_descent_improved(init_p, step_size, epsilon, *args):
    iterates = []
    beta = beta_init
    grad = compute_grad(beta, args[0], args[1], args[2])
    while np.linalg.norm(grad) > epsilon:
        new_beta = beta - step_size*compute_grad(beta, args[0], args[1], args[2])
        iterates.append(new_beta)
        beta = new_beta
        grad = compute_grad(beta, args[0], args[1], args[2])
    return iterates
